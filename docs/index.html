Anki Add-on

AnkiDeck TTS

Generate note audio from text with per-row queue progress and sequential processing.

What It Does

  • Processes current or selected notes in batch mode.
  • Writes generated audio into a target field.
  • Supports multiple TTS providers and models.
  • Configurable overwrite and skip rules for safer runs.

Providers

Current configuration includes support for DashScope, OpenAI, and ElevenLabs.
